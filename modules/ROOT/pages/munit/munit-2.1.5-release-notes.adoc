:keywords: munit, 2.1.5, release notes

This release is focused on fixing bugs and to extend Java support to OpenJDK 11.

== Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8, OpenJDK 11
* Maven 3.3.3, 3.3.9, 3.5.4, 3.6.0

== Compatibility

|Mule | Mule 4.1.4 or higher
|Anypoint Studio | Anypoint Studio 7.3.4
|===

== Features and Functionality

=== MUnit Runtime

No new feature was introduced on this release.

== Known Issues

There are no known issues for this release.

== Bug Fixes

* MUnit maven runner doesn't propagate security settings to Mule embedded container
* Nullpointer exception log when having operation inside before/after-test
* Iterable Matchers doesn't work with Iterators
* Problem when configured two http:request mock processors
* If AssertionError is thrown with a cause, test is marked as ERROR

== Enhancements

* Coerce values to Java String for String matchers

== Story

There are no stories for this release.

=== Anypoint Studio Plugin

This release introduces MUnit Studio plugin 2.3.3.
