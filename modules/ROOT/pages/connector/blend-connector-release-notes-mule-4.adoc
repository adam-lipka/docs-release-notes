Support Category: Select

The Anypoint Connector for Blend lets you connect to the Blend Data APIs to support workflows related to Loans, Borrowers, Realtors, Documents & Disclousers. The connector exposes convenient methods to alter data on Blend platform. MuleSoft recommends that you install the latest version of Studio.

== 1.0.0

*Jun 10, 2019*

=== Compatibility

|Anypoint Studio|7.0 and later
|Java|8
|Blend|v2.0.0
|===

=== Features

* Loans - Processing features with GET,POST,DELETE & PUT
* Borrowers - Processing features
* Realtors - Processing features
* Documents & Disclosures - Processing features
