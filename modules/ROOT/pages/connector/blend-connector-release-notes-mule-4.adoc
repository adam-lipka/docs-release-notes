Support Category: Select

The Anypoint Connector for Blend lets you connect to the Blend Data APIs to support workflows for loans, borrowers, realtors, documents, and disclosures. The connector exposes convenient methods to alter data on the Blend platform. MuleSoft recommends that you install the latest version of Anypoint Studio.

== 1.0.0

*June 10, 2019*

=== Features

* Loans - Processing features with GET,POST,DELETE, and PUT.
* Borrowers - Processing features.
* Realtors - Processing features.
* Documents and Disclosures - Processing features.
