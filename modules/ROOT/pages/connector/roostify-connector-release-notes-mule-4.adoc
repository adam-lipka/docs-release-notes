Support Category: Select

The Anypoint Connector for Roostify allows interaction with Roostify using:

* Webhooks, which are callbacks that let you react to activity in Roostify.
* A RESTful web API that lets operations work with data stored in Roostify.

MuleSoft recommends that you install the latest version of Anypoint Studio.

=== Features

This connector provides access to Roostify APIs for these workflows:

* Access Control
* Accounts
* Desktop Underwriter Report
* Documents
* Leads (Draft)
* Loan Applications
* Messages
* Offerings (Draft)
* Reference IDs
* Service
* Settings
* State Changes
* Status Update
* Tasks
* User Management
* User Settings
* Webhook events
* Webhooks
