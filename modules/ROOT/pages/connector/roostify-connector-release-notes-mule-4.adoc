Support Category: Select

==== The Anypoint Connector for Roostify allows interaction with Roosity in two ways: 
* Webhooks, which are callbacks that allow your systems to react to activity in Roostify.
* A Restful Web API, which allow operations on data stored with Roostify.

MuleSoft recommends that you install the latest version of Studio.

=== Features

MuleSoft Roostify connector offers simplified access to 46 of Roostify's APIs to support workflows related to:

* Loan Applications
* Reference Ids
* Tasks
* Documents
* Messages
* Accounts
* Settings
* Service
* User Settings
* Webhooks
* Webhook events
* Status Update
* User Management
* State Changes
* Desktop Underwriter Report
* Access Control
* Leads (Draft)
* Offerings (Draft)
