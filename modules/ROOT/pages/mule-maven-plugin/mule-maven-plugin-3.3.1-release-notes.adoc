Mule Maven Plugin 3.3.1 is mainly focused on fixing bugs.

== Hardware and Software Requirements

Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8
* Maven 3.3.3, 3.3.9, 3.5.0, 3.6.0

== Compatibility

* Mule 4.x
* Anypoint Studio 7.x
* OpenJDK 11, when using Mule 4.2

== Fixed Issues

* Fixed security vulnerabilities on plugin dependencies.
* Fixed memory and CPU limit settings on RTF deployment
