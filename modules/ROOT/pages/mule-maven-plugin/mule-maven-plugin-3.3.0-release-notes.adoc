Mule Maven Plugin 3.3.0 is mainly focused on fixing bugs.

== Hardware and Software Requirements

Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8 and extended support to OpenJDK 11
* Maven 3.3.3, 3.3.9, 3.5.0, 3.6.0

== Compatibility

* Mule 4.x
* Anypoint Studio 7.x

== Fixed Issues

* CH undeploy need to delete app from the platform too
* Dependencies with different classifier are considered equal
* Validate mule configuration files on packaging
